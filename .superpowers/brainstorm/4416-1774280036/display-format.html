这个 HUD 的展示形式是什么？

选择最符合你设想的展示方式

A

独立 Web 界面（浏览器 HUD）

Claude Code 运行时，浏览器自动弹出一个实时看板，展示 agent 调用树、工具执行状态、skill 流转等信息。类似 Webpack Dashboard 或 Vite 的浏览器 UI。

● Agent: code-reviewer
✓ Tool: Read (src/index.ts)
✗ Tool: Bash (npm test) — failed
● Skill: superpowers:brainstorming
… running
B

终端侧边栏（终端内 TUI 分屏）

在终端右侧实时渲染一个文字面板，与 Claude Code 并排，无需离开终端环境。类似 tmux 分屏 + rich-TUI。

claude-code>
Let me read the file...
HUD
▶ Read
▶ Agent
✗ Bash
C

VS Code / Cursor 侧边栏扩展

在 IDE 的 Activity Bar 中新增一个面板，展示 Claude Code 活动。适合在编辑器中工作时直接查看。

◈ CLAUDE CODE HUD
● Tool Calls (3)
● Active Agents (1)
● Skills (2)
